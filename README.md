# VibeMaker ferret one セクションHTML

ferret one貼り込み用に、各セクションをインラインCSS付きHTML断片として分割しています。

## プレビュー

- 全セクション一覧: `preview_all_sections.html`
- GitHub Pages公開時の入口: `index.html`

このリポジトリは静的HTMLのみで構成されています。ビルド手順は不要です。
