WHAT IS VIBE

TEXT is KING

テキストを制することで
Vibeを、AIが動ける設計図に変える

コードは書かない。
テキストが最大の武器。

Markdownとプロンプトで
AIを動かす。

Cursor を
標準IDEとして活用。

ガチエンジニアは対象外。
非エンジニアが主役。

# MARKDOWN

誰が読んでも、AIが読んでも意味が通じる共通フォーマット。段落・見出し・コードブロックが文脈を生みます。

◎ 議事録 → すぐに要約＆タスク化
◎ 要件書 → そのままAIが実装を提案
◎ 経営資料 → 何度でも再利用できるテンプレ化

@ CONTEXT

営業の暗黙知、人事の温度感、経営の勘所。全部をテキストで残すから、AIが瞬時に参照できます。

◎ Slack・Notion・メールを一元化
◎ 過去プロジェクトの経緯を一瞬で再現
◎ AIが最適な議事録・施策案を提案

& CHAIN

ログとデータが途切れないことが、プロジェクトを守ります。AIがどこからでもバトンを受け取れる状態へ。

◎ Cursorでローカル作業 → ルール同期
◎ GitHub / Box / Dropbox等の既存ポリシー準拠
◎ BS/PLもローカルで安全に分析し制御
